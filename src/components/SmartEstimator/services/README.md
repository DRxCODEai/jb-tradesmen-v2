# Smart Estimator Service Profiles

Future service profiles will be organized into these folders:

- `interior`
- `painting`
- `flooring`
- `tile`
- `plumbing`
- `electrical`
- `hvac`
- `appliances`
- `exterior`
- `maintenance`
- `commercial`
- `remodeling`
- `assessments`
- `emergency`

Every future service profile must satisfy `MasterServiceTemplate`. No production service profiles are included in Phase 1.
