# Smart Estimator Engines

Future deterministic engines will include:

- Pricing Engine
- Timeline Engine
- Confidence Engine
- Scope-of-Work Engine
- Recommendation Engine
- Estimate Summary Engine

Engines must not contain React UI or call external AI services. They must reference approved company standards and registered service profiles. Given the same versioned inputs, standards, and service profile, an engine must produce the same result.

AI may later assist with interpreting customer input, but it cannot set final prices. Final pricing remains deterministic and requires JBTRADESMENLLC approval.
